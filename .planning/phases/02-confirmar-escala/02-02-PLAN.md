  <action>
**Parte 1 — Corrigir o cálculo de $statusClass (~linha 355):**

Localizar esta linha no loop de participantes:
```php
$statusClass = 'status-' . ($member['is_confirmed'] ? 'confirmed' : ($member['status'] == 'declined' ? 'declined' : 'pending'));
```

Substituir por (usa `$member['status']` diretamente):
```php
$memberStatus = $member['status'] ?? 'pending';
// Normalizar: 'absent' exibe como 'declined' visualmente
$statusClass = in_array($memberStatus, ['confirmed', 'declined', 'absent']) ? 'status-' . $memberStatus : 'status-pending';
```

**Parte 2 — Atualizar o indicador circular no avatar:**

Localizar a linha com `<div class="status-indicator <?= $statusClass ?>"></div>` dentro do loop (já existe ~linha 373). Verificar que a classe CSS agora vem de `$statusClass` calculado acima — deve funcionar sem alteração adicional desde que $statusClass seja calculado corretamente.

**Parte 3 — Adicionar badge de texto legível no `.member-info`:**

Dentro do loop, localizar o bloco:
```html

</div>
```

<div class="member-status-badge <?= $statusClass ?>">
